Hi! welcome to Sorte.Io.

We develop this solution to make drawings more simple and efficient.

In a home page you will find 2 buttons that you can choose 2 types of draw:

  • Inputs list draw
  • Upload file draw

INPUT LIST TO DRAW

This type of draw is very simple. Click on "Input list" button. In the next page you'll see a text block. Inside the text block you will type (or paste) all values that you want to draw (e.g. Names, colors, cars, any value) separated by a semicolon. Then you just need to click in "Submit list" and the result will apear bellow. PS: We need it to be separated by semicolon, because is how our system reads and separates values. We are currently working on features to make this easier.

UPLOAD FILE TO DRAW

Click on "Upload file" button. You will find a page that you can upload a files that with the data you want to draw. You just need to click the upload button, find your file and click in "Submit".The result will apear bellow. For now we need a .csv file and you just need to send the first column filled with your values ​​to draw (without column title).
